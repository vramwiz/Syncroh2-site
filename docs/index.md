# 新旧朗2

AviUtl2でPSDファイル形式の立ち絵が使えるようになります。
AviUtl2のプラグインとAviUtl2の操作を補助するアプリで構成されています。

## ✨ 主な機能

- Aviutl2でPSDファイルが扱えます※D&Dは対象外
- PSD立ち絵から表情を自動検出しリストにします（ANMファイルは不要）
- PSDToolKit形式の必要がありません、自動的に解析します。
- 1つのファイルに複数のキャラがあってもただしく表示出来ます。
- AviUtl2上で設定した座標を保存できるので毎回設定は不要です。
- 目パチ、口パク（ソング、トーク）に対応しています。
- 表情を有効にするとAviUtl2上のスライダーで表情指定が出来ます。
- レイヤー状態を設定した状態でD&Dするとその状態の立ち絵になります。
- サムネイル作成用に画像形式でD&Dも可能です。
- 立ち絵のレイヤーとは別に服装レイヤー表情レイヤーの指定が出来ます。
- キャッシュにより初回表示以降は高速に描画エンコードされます。
- セリフを管理する機能があります。
- セリフは音声合成ソフトを監視して取り込まれます
- 取り込んだデータはキャラも登録され送信先レイヤーを割り当てることが出来ます。
- キャラごとに音量とパン（位置）が指定できます。
- 取り込んだセリフはD&Dや送信でAviUtl2へ送ることが出来ます。
- 送信したセリフはアプリからセリフの変更が可能です。
- 専用エクスプローラーを使うと音量やフェードインアウト情報込みのデータを送ることが出来ます。
- その他音楽関係のプラグインやツールが充実しています。

## 🚀 起動方法
### 🔹 自動インストール版
Windowsメニューの新旧朗2から起動します。

### 🔹 手動インストール版
Syncroh_Desktopを実行します。

### 🔹 Aviutl2からの起動方法
- どちらのインストール方法でもAviUtl2から起動出来ます。
- 編集→プラグイン→新旧朗2

### 🔹 Aviutl2の専用ツールバーからの起動方法
- 表示→新旧朗2　とすると専用ツールバーが表示されます
- ツールバーから各種ツールを起動することが出来ます。

## 📌 注意点

- D&Dなど一部の操作において入力プラグイン優先度が下の方だと正常に処理出来ませんので優先順位を上げて対応してください。
- PSDファイル表示も全て自作のため表示出来ない立ち絵がある場合はお気軽に報告してください。可能な限り対応します。
