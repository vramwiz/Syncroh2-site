## 🧱 インストールファイル構成

## インストール先フォルダについて

新旧朗2で使用する **AviUtl2用プラグインおよびスクリプトのインストール先** は、  
AviUtl2 本体の仕様により、以下のフォルダに固定されています。

C:\ProgramData\aviutl2

このフォルダは **AviUtl2が公式に使用している共通データフォルダ** であり、  
プラグイン（.auf2 / .aui2 など）やスクリプト（.lua / .mod2 等）は  
原則としてこの配下に配置されます。

### 隠しフォルダに関する注意

C:\ProgramData は **Windows の既定設定では「隠しフォルダ」** となっているため、  
エクスプローラー上では通常表示されません。

フォルダを直接確認・操作する場合は、  
エクスプローラーの表示設定で **「隠しファイル・隠しフォルダを表示」** を有効にしてください。

※ この挙動は Windows 側の仕様であり、新旧朗2固有のものではありません。

## プラグイン／スクリプトの配置場所について

新旧朗2で使用する各種ファイルは、  
AviUtl2 の仕様に従い C:\ProgramData\aviutl2 配下の専用フォルダに配置します。

AviUtl2 の基本的なフォルダ構成には、次のようなフォルダがあります。

- Plugin
- Script

これらは役割が明確に分かれており、配置先を誤ると AviUtl2 から認識されません。

### Plugin フォルダについて

配置先  
C:\ProgramData\aviutl2\Plugin

このフォルダには、AviUtl2 が直接読み込むプラグイン本体を配置します。

- フィルタプラグイン（.auf2）
- 入力プラグイン（.aui2）
- その他 AviUtl2 が読み込むバイナリ形式のプラグイン

新旧朗2に含まれるプラグイン形式のファイルは、  
必ずこの Plugin フォルダ配下に配置してください。

### Script フォルダについて

配置先  
C:\ProgramData\aviutl2\Script

このフォルダには、AviUtl2 がスクリプトとして読み込むファイルを配置します。

- Lua スクリプト（.lua）
- AviUtl2 用スクリプトモジュール（.mod2）
- スクリプトから参照される補助ファイル

新旧朗2に含まれるスクリプト関連ファイルは、  
必ずこの Script フォルダ配下に配置してください。

### 配置先に関する注意

- Plugin と Script は別の仕組みで動作します
- フォルダを間違えて配置した場合、AviUtl2 では認識されません
- ファイルを配置するときは AviUtl2 を終了してください

これらのフォルダ構成および動作は AviUtl2 側の仕様であり、  
新旧朗2 独自のルールではありません。

## Script フォルダ内の配置構成について

新旧朗2のスクリプト関連ファイルは、  
AviUtl2 の Script フォルダ直下ではなく、**専用のサブフォルダを作成して配置**します。

インストール時の初期構成では、以下のようなフォルダを作成します。

Script フォルダ  
C:\ProgramData\aviutl2\Script

その中に、**新旧朗2用のフォルダ「Syncroh2」** を作成します。

C:\ProgramData\aviutl2\Script\Syncroh2

この「Syncroh2」フォルダが、新旧朗2の **スクリプト関連ファイル一式の配置先** となります。

### 配置するファイルについて

作成した Syncroh2 フォルダ内には、次のようなファイルを配置します。

- @Syncroh2_Script.obj2  
- Syncroh2_Module.mod2  

これらは新旧朗2の動作に必要なスクリプト／モジュールファイルであり、  
**必ず Script\Syncroh2 フォルダ内に配置**してください。

### フォルダを分ける理由

Script フォルダ直下には、他のスクリプトやツールのファイルが配置されることがあります。  
そのため、新旧朗2では以下の点を考慮し、専用フォルダ構成を採用しています。

- 他のスクリプトとの混在を防ぐため  
- ファイル管理を分かりやすくするため  
- 将来的なファイル追加・更新に対応しやすくするため  

この構成は、新旧朗2の初期インストール時の標準構成です。

### 注意事項

- フォルダ名は「Syncroh2」を使用してください  
- フォルダ名や階層を変更した場合、正しく動作しない可能性があります  
- ファイルを配置した後は、AviUtl2 を再起動してください  

これらの配置ルールを守ることで、新旧朗2のスクリプト機能が正しく認識されます。

## Plugin フォルダ内の配置構成について

新旧朗2のプラグイン関連ファイルは、  
AviUtl2 の Plugin フォルダ直下ではなく、**専用のサブフォルダを作成して配置**します。

インストール時の初期構成では、次のようなフォルダ構成になります。

Plugin フォルダ  
C:\ProgramData\aviutl2\Plugin

その中に、**新旧朗2用のフォルダ「Syncroh2」** を作成します。

C:\ProgramData\aviutl2\Plugin\Syncroh2

この「Syncroh2」フォルダが、新旧朗2の **プラグイン一式の配置先** となります。

### 配置するファイルについて

作成した Syncroh2 フォルダ内には、  
新旧朗2で使用するプラグインファイルを配置します。

- Syncroh2_xx.auf2  
- Syncroh2_xx.aui2  
- その他 新旧朗2に含まれるプラグインファイル

ファイル名はすべて **Syncroh2_ から始まる名称** になっており、  
新旧朗2関連のプラグインであることが分かるよう統一されています。

### フォルダを分ける理由

Plugin フォルダ直下には、他のプラグインが多数配置されることがあります。  
そのため、新旧朗2では以下の理由から専用フォルダ構成を採用しています。

- 他のプラグインとの混在を防ぐため  
- プラグイン数が増えても管理しやすくするため  
- アンインストールや更新作業を容易にするため  

### 注意事項

- フォルダ名は「Syncroh2」を使用してください  
- プラグインファイルを Plugin フォルダ直下に直接置かないでください  
- フォルダ名や階層を変更した場合、正しく動作しない可能性があります  
- ファイルを配置した後は、AviUtl2 を再起動してください  

この構成は、新旧朗2の初期インストール時における標準構成です。
