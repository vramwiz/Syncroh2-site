## 🧱 インストールファイル構成

## インストール先フォルダについて

新旧朗2で使用する **AviUtl2用プラグインおよびスクリプトのインストール先** は、  
AviUtl2 本体の仕様により、以下のフォルダに固定されています。

C:\ProgramData\aviutl2

このフォルダは **AviUtl2が公式に使用している共通データフォルダ** であり、  
プラグイン（.auf2 / .aui2 など）やスクリプト（.lua / .mod2 等）は  
原則としてこの配下に配置されます。

### 隠しフォルダに関する注意

C:\ProgramData は **Windows の既定設定では「隠しフォルダ」** となっているため、  
エクスプローラー上では通常表示されません。

フォルダを直接確認・操作する場合は、  
エクスプローラーの表示設定で **「隠しファイル・隠しフォルダを表示」** を有効にしてください。

※ この挙動は Windows 側の仕様であり、新旧朗2固有のものではありません。

## プラグイン／スクリプトの配置場所について

新旧朗2で使用する各種ファイルは、  
AviUtl2 の仕様に従い C:\ProgramData\aviutl2 配下の専用フォルダに配置します。

AviUtl2 の基本的なフォルダ構成には、次のようなフォルダがあります。

- Plugin
- Script

これらは役割が明確に分かれており、配置先を誤ると AviUtl2 から認識されません。

### Plugin フォルダについて

配置先  
C:\ProgramData\aviutl2\Plugin

このフォルダには、AviUtl2 が直接読み込むプラグイン本体を配置します。

- フィルタプラグイン（.auf2）
- 入力プラグイン（.aui2）
- その他 AviUtl2 が読み込むバイナリ形式のプラグイン

新旧朗2に含まれるプラグイン形式のファイルは、  
必ずこの Plugin フォルダ配下に配置してください。

### Script フォルダについて

配置先  
C:\ProgramData\aviutl2\Script

このフォルダには、AviUtl2 がスクリプトとして読み込むファイルを配置します。

- Lua スクリプト（.lua）
- AviUtl2 用スクリプトモジュール（.mod2）
- スクリプトから参照される補助ファイル

新旧朗2に含まれるスクリプト関連ファイルは、  
必ずこの Script フォルダ配下に配置してください。

### 配置先に関する注意

- Plugin と Script は別の仕組みで動作します
- フォルダを間違えて配置した場合、AviUtl2 では認識されません
- ファイルを配置するときは AviUtl2 を終了してください

これらのフォルダ構成および動作は AviUtl2 側の仕様であり、  
新旧朗2 独自のルールではありません。
